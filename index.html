☁️ Configuración Firebase (Sincronización)
📋 Pasos para obtener tu configuración Firebase:
1. Ve a console.firebase.google.com
2. Clic en "Agregar proyecto" → nombre: Asistencia JTB → desactiva Analytics → Crear
3. En el menú izquierdo: Compilar → Realtime Database → Crear base de datos
4. Elige ubicación us-central1 → Modo de prueba → Habilitar
5. Clic ⚙️ junto a "Descripción general del proyecto" → Configuración del proyecto
6. Baja hasta "Tus apps" → clic ícono </> → pon nombre JTBApp → Registrar
7. Copia todo el objeto firebaseConfig = { ... } y pégalo abajo
⚠️ Reglas de seguridad: En Firebase → Realtime Database → Reglas, asegúrate de tener:
{ "rules": { ".read": true, ".write": true } }
